# Automobile Dataset - Exploratory Data Analysis (EDA)

## Overview

The **Automobile Dataset - Exploratory Data Analysis (EDA)** project explores an automobile dataset to uncover insights into vehicle characteristics, pricing, fuel efficiency, and manufacturer trends. Using Python and data visualization techniques, the project performs data cleaning, statistical analysis, and visual exploration to identify relationships between key vehicle attributes.

This project demonstrates practical data analysis skills by transforming raw automotive data into meaningful insights that can support data-driven decision-making.

---

## Objectives

- Explore the automobile dataset and understand its structure
- Clean and preprocess the data
- Handle missing values and inconsistent data
- Analyze relationships between vehicle specifications and price
- Identify trends across manufacturers
- Visualize key findings using charts and graphs

---

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ghamandiashwini/Automobile_Dataset_EDA.git
```

2. Navigate to the project folder

```bash
cd Automobile_Dataset_EDA
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
